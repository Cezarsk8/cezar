# Cezar Sevilhano - Portfolio Website

Personal portfolio website showcasing professional experience, projects, and skills.

## 🚀 Live Demo

Visit: [cezarsk8.github.io/cezar](https://cezarsk8.github.io/cezar/)

## ✨ Features

- **Hero Section**: Full-viewport with typing animation and key metrics
- **Projects Showcase**: Listo (price tracking) and InCasa Fitness (workout management)
- **Skills**: Animated progress bars with technology tags
- **Experience Timeline**: Career history with achievements and tech stack
- **Contact Form**: Formspree integration with validation
- **Responsive Design**: Mobile-first with 4 breakpoints (480px, 768px, 1024px, 1440px)
- **Accessibility**: ARIA labels, focus states, reduced motion support
- **SEO**: Meta tags, Open Graph, JSON-LD structured data
- **Print Stylesheet**: Optimized for CV printing

## 🛠️ Tech Stack

| Category | Technology |
|----------|------------|
| Markup | HTML5 |
| Styling | CSS3 with CSS Variables |
| JavaScript | Vanilla JS (Intersection Observer, Fetch API) |
| Fonts | Google Fonts (Roboto, Open Sans) |
| Icons | Font Awesome 6.0 |
| Forms | Formspree |
